# 🎬 Buscador de Filmes em Java

Sistema simples desenvolvido em Java que permite ao usuário buscar filmes pelo nome e descobrir em qual plataforma de streaming estão disponíveis.

---

## 📌 Sobre o projeto

Este projeto foi criado com o objetivo de praticar conceitos básicos de programação em Java, como:

- Programação orientada a objetos (POO)
- Uso de listas (`ArrayList`)
- Estruturas de repetição (`for-each`)
- Entrada de dados com `Scanner`
- Manipulação de Strings

---

## 🚀 Funcionalidades

- 🔍 Buscar filmes pelo nome
- ✂️ Busca parcial (não precisa digitar o nome completo)
- 🔡 Ignora letras maiúsculas/minúsculas
- 📺 Exibe onde o filme está disponível (streaming)
- ⚠️ Mensagem caso o filme não seja encontrado

---

## 💻 Tecnologias utilizadas

- Java
- IDE: Eclipse (ou qualquer outra)
