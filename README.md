# 🎬 Buscador de Filmes em Java

Sistema simples desenvolvido em Java que permite ao usuário buscar filmes pelo nome e descobrir em qual plataforma de streaming estão disponíveis.

---

## 📌 Sobre o projeto

Este projeto foi criado para praticar conceitos básicos de Java, como:

- Programação orientada a objetos (POO)
- Uso de ArrayList
- Estruturas de repetição (for-each)
- Entrada de dados com Scanner
- Manipulação de Strings

---

## 🚀 Funcionalidades

- Buscar filmes pelo nome  
- Busca parcial (não precisa digitar completo)  
- Ignora maiúsculas/minúsculas  
- Exibe onde assistir  
- Mensagem caso não encontre  

---

## 💻 Tecnologias

- Java

---

## ▶️ Como executar

1. Clone o repositório: https://github.com/luizthefisher-alt/buscador-filmes-java.git
2. Abra o projeto em uma IDE (Eclipse ou IntelliJ)
3. Execute a classe `Main.java`
